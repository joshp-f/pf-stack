# PF Stack

A full-stack component library built to accelerate copilot product development across frontend, ai, infrastructure and engineering. 

- As open source as possible. No affiliation or lock in with vendors.
- Fully modular. Use only what you need. No lock in to PF stack.
- Quick start presets for demos to production ready
- Full SOTA capabilities matching Industry SOTA
